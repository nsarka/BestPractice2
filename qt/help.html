BestPractice

Open and play audio

Open or drop a WAV or MP3 file. Use the transport controls to play, pause, restart, seek, and set volume.

Time and pitch

Speed changes duration without changing pitch. Coarse and fine pitch controls change pitch without changing duration. Higher time-stretch quality uses more processing.

Looping

Set Start and End with the time controls or the Now buttons, then enable Loop. Saving while Loop is enabled exports only that interval. Processed audio can be saved as lossless WAV or as MP3 at a selected bit rate.

Karaoke

Karaoke mode suppresses centered vocals. Vocal position and bass/treble pass-through can preserve instruments that would otherwise be removed.

Keyboard

Numpad 0 plays or pauses. Backspace restarts. F5 increases speed and Shift+F5 decreases it. F1 opens this help.
