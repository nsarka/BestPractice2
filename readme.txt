licensed under the GPL v2. See the file 'license.txt' for details.

You will need Borland C++ Builder 6 to be able to build BestPractice
from source code. 

Modernization work has started in parallel with the original Borland project.
The portable audio core now has a CMake build, and the VCL forms have a Qt 6
Widgets replacement under the qt/ directory. See MODERNIZATION.md for details.

The installer can be built using the bp.nsi script for the NSIS installer.

The pop-up help files is a ShalomHelp project whereas the htmlhelp project
is a vizacc HelpMaker project.

You will need to install the Windows XP Theme Manager for Delphi and Borland 
C++  Written by Mike Lischke, since it is used by BestPractice. See www.soft-gems.net.

If you use code from BestPractice, a small note somewhere, for instance
the about box, is appreciated but not necessary as long as the conditions
as specified in the GPL license are fulfilled.

The project's home page is http://bestpractice.sourceforge.net
